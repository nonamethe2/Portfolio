Caravan (Simplified) Info

Description

Caravan originally appeared in the 2010 video game Fallout: New Vegas, developed by Obsidian Entertainment, and features many complex and often arbitrary rules. The basic premise of the game is to sell your caravans. The caravans being stacks of cards that has a value determined by which cards are added. This comes with various restrictions governing what makes a caravan sellable, but gameplay mostly consists of trying to add or subtract to the value of your caravans to sell your caravans before your opponent manages to do the same.

Due to time restraints I decided to modify the rules of the game to make it simpler and easier to play. I modified a majority of the rules, going so far as to remove certain cards from play entirely. I ensured that all my alterations to the rules did not drastically alter gameplay or the end goal so that at its core my version is still the same game. One feature I added to my version is single player gameplay where the user plays against the computer.

My implementation of this card game was done entirely in Java. I had originally written this for a class which had as the final project assigned a particularly broad subject which allowed me to build such a large application when the project itself did not need to be of this scale.
