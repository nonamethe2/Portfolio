Using Custom Textures

Info

A number of the hats in the game currently support texture swapping. You are able to use your own texture on any of the hats that support this. On this page you will see in order: a list of hats that support this feature, instructions on how to add the texture to the game files, and finally the UV maps that were used.

Hats that support this

  • Baseball Cap*
  • Beanie Hat*
  • Bowler Hat*
  • Cow Boy Hat**
  • Sun Hat*
  • Party Hat
  • Wizard Hat

The UV map determines the way in which the texture gets mapped to the hat and so for reference I have uploaded the UV maps that I generated when creating the hats. To download a UV map just click on the link for the corresponding hat that you want to apply a texture for.

You don't need to include the UV map in the texture file you insert into the textures folder; it's only use is to provide a reference for how your texture will be mapped onto the hat.

How to Use

How to use: In the root directory of the game files, which is in the same folder as the executable there should exist a folder named "Saves", if this folder does not exist you can create it or just run the game and it will be created. Then within that folder add a folder named "Textures" but without the quotes.

The above only applies if you're running the game on Windows, Mac users will have to check their ~/Library folder.

The script is set up to take png images that use the hats name for identification plus the word Retex. Some hats actually use two separate textures, in this case append " Main" or " Brim" to the name of the file. The baseball cap for instance comprises of one texture for the main part that covers the head, while the brim of the hat is a separate texture. These hats are denoted with an "*" after the name.

It is also important that the files have the .png extension and not the .jpg extension.
